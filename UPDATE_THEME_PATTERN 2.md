# Patrón de actualización de tema para componentes de consulta

Los componentes de consulta de usuarios deben seguir este patrón:

1. Cambiar container a min-h-screen con fondo degradado
2. Header con gradiente usando var(--color-primary)
3. Tabs modernos con bordes y colores del tema
4. Tablas con thead con gradiente suave
5. Botones con colores del tema
6. Modales con header con gradiente

Ver consultar-gerente.component.html como referencia.
